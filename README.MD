- fly.js拦截器
- 多环境参数配置

> 相关文章：微信小程序之wepy自动化架构搭建（fly+wepy-plugin-replace）

## 部署说明

### 安装环境
```
# 安装（更新） wepy 命令行工具。
cnpm install wepy-cli -g
# 安装依赖包
cnpm install
# 开发实时编译。
npm run dev
# 打包。
npm run build
```

### 小程序工具
- 使用微信开发者工具-->添加项目，选择dist目录；
- 可以使用体验APPID；
- 关闭ES6转ES5；
- 关闭上传代码时样式自动补；
- 关闭代码压缩上传；
- 打开不校验合法域名；

> [WePY开发文档参考](https://tencent.github.io/wepy/)
